De onde vêm os nossos visitantes?

Participa num inquérito global aos visitantes do stand da INETE. Selecciona no mapa a tua freguesia de origem e escolhe o número de visitantes de acordo com o número de pessoas no teu grupo. Submete o número de visitantes e aprecia os resultados!

Esta aplicação exemplifica, de forma simples, o potencial dos Sistemas de Informação Geográfica.

Tecnologias utilizadas:

  • A interface gráfica é implementada utilizando HTML + CSS + Javascript, as linguagens da Web. São utilizadas as bibliotecas:
    • JQuery
    • OpenLayers
    • flot
  • O mapa:
    • Utiliza dados de base provenientes da Carta Administrativa Oficial de Portugal;
    • É disponibilizado através do standard Web Map Service.
    • É servido para a web utilizando a aplicação MapServer.
  • O servidor é composto por dois componentes:
    • Uma base de dados espacial Spatialite, que serve os dados ao servidor de mapas e permite o armazenamento do número de visitantes;
    • A plataforma Geodjango, que fornece a um conjunto de funcionalidades adequadas ao à criação de aplicações web para informação geográfica.
